# Java Backend Learning Journey ☕️

Welcome to my Java Backend Learning Journey repository.

This repository serves as a public record of my journey from learning basic Java programming concepts to building complete backend applications using Spring Boot. Rather than following a purely theory-based approach, I am learning through projects. Every project in this repository represents a stage in my growth as a Java developer.

The goal is simple:

> Learn → Build → Improve → Repeat

---

# 🎯 Purpose of This Repository

The purpose of this repository is to:

- Learn Java from the ground up.
- Understand Object-Oriented Programming through practical implementation.
- Gain hands-on experience with real-world software development.
- Learn backend development concepts progressively.
- Build a portfolio that demonstrates consistent learning and improvement.
- Transition from Java fundamentals to enterprise application development with Spring Boot.

---

# 📖 Learning Methodology

Instead of only watching tutorials or reading documentation, I follow a project-based learning approach.

For every project:

1. A project idea is selected.
2. A Software Requirements Specification (SRS) document is prepared.
3. The application is designed and developed using the concepts learned so far.

This approach helps transform theoretical knowledge into practical development experience.

---

# 🤖 AI-Assisted Development

All project ideas and initial SRS documents are generated with the assistance of AI.

For each project folder, you may find:

- Software Requirements Specification (SRS)
- Source Code
- Result ScreenShots

While AI assists with ideation and requirement generation, the actual learning, coding, debugging, implementation, and understanding are part of my own development process.

The goal is not simply to complete projects but to understand why and how they work.
